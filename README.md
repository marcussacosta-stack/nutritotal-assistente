# Nutritotal Assistente

Assistente virtual de nutrição que ajuda a criar planos alimentares personalizados, listas de compras e acompanhar o progresso.

## Funcionalidades

- **Criação de Planos Alimentares**: Gere planos semanais baseados em preferências.
- **Lista de Compras**: Gere listas de compras automáticas a partir dos planos.
- **Rastreamento de Progresso**: Acompanhe peso e ingestão de água.
- **Assistente IA**: Converse com um assistente especializado em nutrição.

## Tecnologias

- React
- TypeScript
- Vite
- Supabase
- Google Gemini AI

## Como Iniciar

1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Configure as variáveis de ambiente em `.env.local`.
4. Inicie o servidor de desenvolvimento: `npm run dev`
